# Cardamom House Landing Page

A modern, responsive landing page built as part of a Frontend Developer trial task. The project focuses on creating a clean user experience with smooth interactions, responsive layouts, and maintainable React components.

## 🌐 Live Demo

https://cardamom-house-seven.vercel.app/

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Modern UI with smooth animations
- Reusable React components
- Clean and organized project structure
- Optimized performance
- Accessible and semantic HTML
- Fast development with Vite

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

Navigate to the project folder:

```bash
cd <repository-name>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
